# UI快速启动指南

## 🚀 快速开始

### 1. 安装依赖
```bash
pip install -r requirements.txt
```

### 2. 启动后端API（可选）
```bash
# 在一个终端窗口中
python -m uvicorn src.api.main:app --reload --port 8000
```

### 3. 启动Streamlit UI
```bash
# 在另一个终端窗口中
streamlit run src/ui/app.py
```

### 4. 访问应用
- 主应用：http://localhost:8501
- API文档：http://localhost:8000/docs

## 📋 功能页面

### 1. 📝 JD分析
**用途**：分析单个JD文件或文本
- 输入JD文本或上传文件
- 选择评估模型
- 查看解析结果、质量评分和优化建议

### 2. 📤 批量上传
**用途**：批量处理多个JD文件
- 一次上传最多20个文件
- 实时查看处理进度
- 查看批量处理结果汇总

### 3. 🗂️ 职位分类管理
**用途**：管理职位分类体系
- 创建3层级分类树
- 为第三层级添加样本JD
- 编辑和删除分类

### 4. 📋 问卷管理
**用途**：生成和管理评估问卷
- 基于JD自动生成问卷
- 预览和编辑问卷
- 生成分享链接

### 5. 🎯 匹配结果
**用途**：查看候选人匹配结果
- 查看匹配度分数
- 维度得分雷达图
- 优势和差距分析

### 6. 📄 模板管理
**用途**：管理解析、评估和问卷模板
- 创建自定义模板
- 编辑模板配置
- 使用预置模板

### 7. 📚 历史记录
**用途**：查看所有分析历史
- 浏览历史记录
- 快速查看结果

### 8. 📝 问卷填写（独立页面）
**用途**：候选人填写问卷
- 访问：http://localhost:8501/questionnaire?id={questionnaire_id}
- 填写问卷题目
- 查看匹配结果

## 💡 使用技巧

### JD分析
1. 可以使用"加载示例JD"快速体验
2. 支持TXT、PDF、DOCX格式文件
3. 结果分为3个标签页：解析、评估、建议

### 批量上传
1. 选择多个文件后会显示文件列表
2. 系统会验证文件大小和数量
3. 处理完成后可以展开查看每个JD的详情

### 职位分类
1. 必须按层级顺序创建（先1级，再2级，最后3级）
2. 只有第三层级可以添加样本JD
3. 删除分类前需要先删除子分类

### 问卷生成
1. 需要先有已分析的JD
2. AI会根据JD自动生成5-10个问题
3. 分享链接可以直接发送给候选人

### 匹配结果
1. 结果按JD分组显示
2. 点击"查看详情"可以看到完整的匹配分析
3. 支持下载HTML和JSON格式报告

## 🔧 配置

### API基础URL
默认：`http://localhost:8000/api/v1`

修改方式：
```bash
# 设置环境变量
export API_BASE_URL=http://your-api-url/api/v1
```

### 评估模型
在侧边栏选择：
- 标准评估
- 美世国际职位评估法
- 因素比较法

## ⚠️ 注意事项

### 文件上传限制
- 单个文件最大：10MB
- 批量上传最多：20个文件
- 总大小限制：100MB

### 支持的文件格式
- TXT：纯文本文件
- PDF：PDF文档
- DOCX：Word文档（2007及以上）

### 浏览器兼容性
推荐使用：
- Chrome（最佳）
- Firefox
- Edge
- Safari

## 🐛 常见问题

### Q: 启动后无法访问？
A: 检查端口8501是否被占用，可以使用 `--server.port` 参数指定其他端口

### Q: API请求失败？
A: 确保后端API服务已启动（http://localhost:8000）

### Q: 文件上传失败？
A: 检查文件大小和格式是否符合要求

### Q: 问卷链接无法访问？
A: 确保问卷ID正确，格式为：`/questionnaire?id=quest_xxxxx`
