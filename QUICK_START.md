## 快速测试（5分钟）

### 1️⃣ 测试 JD 解析（2分钟）

1. 点击侧边栏 **"📝 JD解析（第一步）"**
2. 选择 **"标准解析模板"**
3. 点击 **"📄 加载示例JD"**
4. 点击 **"🔍 解析并保存"**
5. ✅ 查看解析结果

### 2️⃣ 测试 JD 评估（2分钟）

1. 点击侧边栏 **"⭐ JD评估（第二步）"**
2. 选择刚才保存的 JD
3. 选择 **"标准评估"** 模板
4. 点击 **"⭐ 提交评估"**
5. ✅ 查看评估结果（质量分数、企业价值、核心岗位判断）

### 3️⃣ 测试企业管理（1分钟）

1. 点击侧边栏 **"🏢 企业管理"**
2. 在右侧输入企业名称（如："科技有限公司"）
3. 点击 **"✅ 创建企业"**
4. ✅ 查看企业列表

## 常见问题

### Q: 刷新页面后数据丢失了？
**A**: 这是正常的。当前使用 session_state 存储数据，刷新会清空。建议不要刷新页面，使用侧边栏导航。

### Q: 看到"API请求失败"提示？
**A**: 运行 `python check_api.py` 检查 API 状态。如果 API 未运行，执行 `python -m src.api.main`。

### Q: 如何创建职位分类？
**A**: 
1. 先在"企业管理"创建企业
2. 进入"职位分类管理"
3. 选择企业
4. 创建三层级分类（大类→中类→小类）
5. 为第三层级添加标签
