# Streamlit UI - 岗位JD分析器前端

## 概述

基于Streamlit构建的Web界面，提供完整的JD分析、批量上传、职位分类管理、问卷生成和匹配评估功能。

## 功能页面

### 1. 📝 JD分析页面
- 文本输入或文件上传（TXT、PDF、DOCX）
- 实时AI分析和质量评估
- 解析结果、质量评分、优化建议展示
- 职位分类展示和手动调整

### 2. 📤 批量上传页面
- 支持批量上传最多20个JD文件
- 实时进度显示
- 批量处理结果汇总
- 成功/失败文件列表
- 每个JD的快速预览

### 3. 🗂️ 职位分类管理页面
- 3层级分类树展示
- 创建、编辑、删除分类
- 第三层级样本JD管理（1-2个）
- 分类树可视化

### 4. 📋 问卷管理页面
- 基于JD自动生成评估问卷
- 问卷预览和编辑
- 分享链接生成
- 问卷列表管理

### 5. 🎯 匹配结果页面
- 匹配度分数可视化
- 维度得分雷达图
- 优势和差距分析
- 候选人排名列表
- 报告下载（HTML/JSON）

### 6. 📄 模板管理页面
- 解析模板、评估模板、问卷模板
- 创建和编辑模板
- 模板列表展示

### 7. 📚 历史记录页面
- 所有分析记录
- 快速查看历史结果

### 8. 📝 问卷填写页面（独立页面）
- 候选人填写问卷
- 实时匹配评估
- 结果展示

## 启动方式

```bash
# 启动Streamlit应用
streamlit run src/ui/app.py

# 或使用启动脚本
python run.py
```

## 依赖项

- streamlit
- requests
- pandas
- plotly

## API集成

所有页面通过REST API与后端通信：
- API基础URL: `http://localhost:8000/api/v1`
- 支持的端点：JD分析、批量上传、分类管理、问卷、匹配、模板
